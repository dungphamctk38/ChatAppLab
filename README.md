# Week10.ChatAppLab

Giai phap WinForms `.NET Framework 4.8` gom hai ung dung rieng:

- `ChatServerApp`: mo server va cho client ket noi.
- `ChatClientApp`: ket noi vao server va gui nhan tin nhan.

Code duoc viet theo kieu don gian, khong dung `async/await`.

## Cach chay

1. Mo solution `Week10.ChatAppLab.sln`.
2. Chay `ChatServerApp`, giu port `9000`, sau do bam `Start Server`.
3. Chay `ChatClientApp`, giu host `127.0.0.1` va port `9000`, sau do bam `Connect`.
4. Gui tin nhan o ca hai app de chat qua localhost.

## Ghi chu

- Hai app la hai file `.exe` rieng, phu hop voi yeu cau start rieng.
- Giao dien server duoc giu gan voi hinh mau ban gui.
